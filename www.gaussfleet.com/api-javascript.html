Inspire-se e crie funcionalidades baseadas em nossa API

Sobre

Nós disponibilizamos nossa API para construção de telas com o seu time de desenvolvimento, te dando flexibilidade e velocidade em criação de novas features sem depender do nosso time.

Como Utilizar

Para utilizar nossa API é simples, basta gerar uma CHAVE em nossa plataforma ou, caso não tenha acesso, solicitar para o responsável que possui acesso, para gerar uma CHAVE DE API no menu Administração da plataforma.

Tendo a Chave em mãos, acesse nossa API utilizando a URL: https://www.gaussfleet.com/api/v1/private/MEU_METODO passando os parâmetros através de campos em POST, para cada função, existirá parâmetros específicos que você pode encontrar na lista abaixo.

  • Versão v1
  • Métodos
  • - Obter Veículo
  • - Estatísticas Veículo
  • - Paradas do Veículo
  • - Estacionamentos do Veículo
  • - Estatísticas Vários Veículos
  • - Paradas Vários Veículos
  • - Estacionamentos Vários Veículos

Obter Veículo (obter_veiculos)

Parâmetro Tipo Obrigatório Descrição
veiculo (string) NÃO Placa do veículo desejado

* Limite de 1 chamada por hora para todos os usuários com acesso a API.
  • Chamada
  • Retorno

    const obter_veiculos = async () =>{
        const URL_AJAX = 'https://www.gaussfleet.com/api/v1/private';
        let data = {
        'veiculo': '(string)',
        }
        const resposta = await fetch( URL_AJAX + '/obter_veiculos/', {
            method: 'POST',
            headers: new Headers({
                'Accept': 'application/json',
                'Content-Type': 'application/json',
                'X-AUTH-TOKEN': 'CHAVE_API'
            }),
            body: JSON.stringify(data)
        }).catch(function(error) {
            throw error;
        });
        return resposta.json();
    }

    planta_operacao_codigo : "", 
 }
